## Environment variables (per project — values from **that** stack only)

Add these **keys**; **values** = whatever you put in the matching GitHub environment (`instance.txt` / secrets).

| Key | Value source |
|-----|----------------|
| `VITE_CONTENTSTACK_API_KEY` | That stack’s API key |
| `VITE_CONTENTSTACK_DELIVERY_TOKEN` | That stack’s delivery token |
| `VITE_CONTENTSTACK_ENVIRONMENT` | Publish env uid (e.g. `dev22-sso-traffic-explorer` or `dev11-traffic-explorer`) |
| `VITE_CONTENTSTACK_DELIVERY_HOST` | That stack’s delivery host URL (no trailing slash) |
| `VITE_CONTENTSTACK_CONTENT_TYPE_UIDS` | Same shared list as GitHub repo variable, e.g. `demo_plain_text,demo_json_rte,demo_reference,demo_group,demo_blocks` |

Do **not** add `CONTENTSTACK_MANAGEMENT_TOKEN` to Launch (not used in the browser build for this app).
